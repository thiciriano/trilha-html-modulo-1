Introdução ao HTML: Os Primeiros Passos

No curso Introdução ao HTML na Prática, aprendemos o básico para criar páginas na internet. Primeiro, conhecemos ferramentas como o Visual Studio Code, que é como um caderno digital para escrever código, e o Inspetor de Elementos, que nos deixa ver como as páginas são feitas.

O HTML tem uma estrutura básica: usamos tags como html, head e body para montar o esqueleto da página. Tags são como etiquetas que dizem o que cada parte faz, como h1 para títulos grandes e p para parágrafos. Algumas tags têm atributos, que são instruções extras, como dizer que um link leva para outro site.

Aprendemos a formatar textos, como usar font (uma tag antiga, hoje menos usada) para mudar a cor, e a fazer listas. Por exemplo, uma lista ordenada (ol) para passos:

  1. Escrever a tag html
  2. Adicionar head e body
  3. Usar h1 para o título principal

E uma lista não ordenada (ul) para ideias soltas:

  • Títulos
  • Textos
  • Links com a

Links (a) são como portas para outras páginas, e usamos atributos para dizer aonde elas levam, como exemplo.com.

HTML é como um mapa que organiza tudo na página da web. - Instrutor do Curso

Formulários: Interagindo com os Visitantes

No curso Trabalhando com Formulários, aprendemos a criar formas de conversar com quem visita nossa página. Usamos a tag form para fazer um formulário, como um questionário. Dentro dele, a tag input cria campos para escrever coisas, como nome ou e-mail. Existem vários tipos de input, como:

  • text: para escrever palavras
  • checkbox: para marcar várias opções, como escolher sabores de pizza
  • radio: para escolher só uma opção, como “sim” ou “não”

Também usamos button para botões, como “Enviar”, e select para listas de escolhas, como um menu de cores favoritas. A tag textarea é para escrever textos maiores, como uma mensagem. A tag font foi mencionada como algo antigo, substituído por outras formas de estilizar.

Estruturação e Formatação: Organizando e Enfeitando

No curso Estruturando seu HTML + Formatações, aprendemos a organizar a página e deixar o texto bonito. Usamos div para criar caixas que agrupam pedaços da página e span para destacar partes pequenas do texto, como uma palavra especial.

Para enfeitar, usamos tags como strong para texto importante, i para itálico, e u para sublinhar. Também usamos sub para escrever coisas como H2O (água) e sup para x2 (elevado ao quadrado).

A tag fieldset organiza formulários, como colocar uma moldura em torno de perguntas. Também vimos embed e iframe para adicionar conteúdos, como vídeos de outros sites, e aprendemos sobre cores para deixar a página mais alegre.

Mídias: Adicionando Imagens, Sons e Vídeos

No curso Trabalhando com Mídias, descobrimos como colocar coisas legais na página, como fotos, músicas e vídeos. A tag img adiciona imagens, e usamos o atributo alt para descrever a imagem, ajudando quem usa leitores de tela. A tag audio toca músicas, e video mostra vídeos.

A tag track adiciona legendas aos vídeos, e iframe é como uma janela para mostrar outras páginas dentro da nossa. Tudo isso faz a página ficar mais divertida!

Tabelas: Organizando Informações

No curso Criando Tabelas, aprendemos a fazer tabelas, que são como grades para organizar informações. Usamos a tag table para criar a tabela, tr para linhas, td para células com dados e th para títulos das colunas. Também usamos thead (cabeçalho), tbody (corpo) e tfoot (rodapé) para dividir a tabela em partes.

Exemplo de coisas que podemos organizar em tabelas:

  • Nomes de amigos
  • Horários de aulas
  • Pontuações de jogos

HTML Semântico: Páginas Inteligentes

No curso Entendendo HTML Semântico, aprendemos a usar tags que deixam a página mais organizada e fácil de entender, tanto para pessoas quanto para computadores. Por exemplo, usamos header para o topo da página, nav para o menu, main para o conteúdo principal, section para dividir partes e footer para o rodapé.

Outras tags, como aside para conteúdos extras, article para textos completos e blockquote para citações, ajudam a organizar tudo. Veja um exemplo:

Uma página semântica é como uma casa onde cada cômodo tem um nome claro, como “cozinha” ou “sala”. - Instrutor

Também aprendemos sobre acessibilidade, para que todos possam usar a página, e SEO (Otimização para Motores de Busca), para ajudar o Google a encontrar nossa página. Tags como figure, figcaption e picture são usadas para imagens especiais, e q para citações curtas.
